Mezclador de sonidos

Open source Software libre
Rust

Estoy desarrollando una aplicación de escritorio que permite manejar el volumen de las diferentes fuentes de audio. Esta herramienta es especialmente útil para usuarios que necesitan controlar múltiples fuentes de sonido simultáneamente, como streamers, DJs, y profesionales del audio. La aplicación está siendo desarrollada en Rust, aprovechando su rendimiento y seguridad para ofrecer una experiencia fluida y confiable. Además, es un proyecto de código abierto, lo que permite a otros desarrolladores contribuir y mejorar la herramienta continuamente.
